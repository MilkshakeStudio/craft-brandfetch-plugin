## Requirements

This plugin requires Craft CMS 3.0.0-beta.23 or later.

## Installation

To install the plugin, follow these instructions.

1. Open your terminal and go to your Craft project:

      `cd /path/to/project`
   
2. Then tell Composer to load the plugin:

      `composer require MilkshakeStudio/brandfetch`

3. In the Control Panel, go to Settings → Plugins and click the “Install” button for Brandfetch.

4. Add your Brandfetch Plugin you need to obtain an API Key from [Brandfetch.io](https://brandfetch.io/) 



## Overview

The Brandfetch Plugin (unofficial) utilizes the API of Brandfetch.io to call and save Logos from just about any URL.  


## Configuring Brandfetch

1. Add API Key to the settings
2. Create a new field with the `Brandfetcher` field type.
3. Enter a URL and voila!
